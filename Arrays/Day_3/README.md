# Day 3 - Linear Search

## Topics Learned

### Linear Search
- Searching an element in an array
- Traversing the array one by one
- Comparing each element with the target

## Variations of Linear Search

1. Search Element
2. First Occurrence
3. Last Occurrence
4. Count Occurrences

## Key Concepts

- Linear Search works on both sorted and unsorted arrays.
- Elements are checked one by one.
- First Occurrence uses break statement.
- Last Occurrence updates index until the loop ends.
- Count Occurrences uses a counter variable.

## Time Complexity

### Best Case
O(1)

(Target found at first position)

### Worst Case
O(n)

(Target found at last position or not present)
