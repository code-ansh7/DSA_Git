 # Day 1 - Time Complexity & Basic Array Problems

## Topics Learned

### Time Complexity
- O(1)
- O(n)
- O(n²)
- O(log n)

### Key Concepts
- Single Loop → O(n)
- Nested Loops → O(n²)
- Doubling Pattern (i *= 2) → O(log n)
- Halfing Pattern (i /= 2) → O(log n)

## Problems Solved

### 1. Array Sum

### 2. Array Minimum
Find the smallest element in an array.

### 3. Array Maximum
Find the largest element in an array.
